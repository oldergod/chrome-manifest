> Help you to manage manifest for developing Chrome Apps and Extensions


## Install

```sh
$ npm install --save chrome-manifest
```


## Usage

```js
var chrome-manifest = require('chrome-manifest');
```

```sh
$ npm install --global chrome-manifest
$ chrome-manifest --help
```
